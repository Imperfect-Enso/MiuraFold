# MiuraFold

MiuraFold is a kinetic building-integrated photovoltaic (BIPV) facade system based on Miura-Ori rigid origami.

The system enables a single-degree-of-freedom deployment mechanism that dynamically adjusts photovoltaic exposure while maintaining compact stowage.
